# Netflix Clone

A responsive Netflix clone website built with HTML and CSS, featuring a modern streaming interface with multiple content categories.

## Features

- **Responsive Navigation Bar**
  - Netflix logo
  - Navigation menu (Home, TV Shows, Movies, New & Popular, My List, Browse by Language)
  - Search functionality
  - Profile dropdown with sign out option
  - Notification bell icon

- **Hero Banner**
  - Featured content display (Money Heist)
  - Play and My List buttons
  - Content description
  - Gradient fade effects

- **Content Categories**
  - Popular on Netflix
  - Trending Now
  - TV Shows
  - Blockbuster Action & Adventure
  - Netflix Originals

- **Footer Section**
  - Social media links (Facebook, Instagram, Twitter, YouTube)
  - Quick links (Help Center, Gift Cards, Jobs, Terms of Use, etc.)
  - Copyright information

## Technologies Used

- HTML5
- CSS3
- Font Awesome 5.1.0 (for social media icons)

## Setup

1. Clone or download the repository
2. Ensure all image assets are in the correct directories:
   - `/assets/` for logos and icons
   - `/netflix card photo/` for content thumbnails
3. Open `index.html` in your web browser

## Dependencies

- Font Awesome 5.1.0 (loaded via CDN)
- External stylesheet: `style.css`

## Browser Compatibility

Works best on modern browsers:
- Chrome
- Firefox
- Safari
- Edge
